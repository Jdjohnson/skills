## Source Discipline

Use `references/*.md` as the standing cache.

Rules:

- Do not invent quotes, publications, positions, dates, or citations.
- Never promote a `TODO:` note from a reference file into output as if sourced.
- If a cached position is missing, either stay silent or label the line as extrapolated from the economist's framework.
- For "latest", stage, article, publication, or quote-sensitive work, do targeted live lookup and cite current sources.
- Keep sourced claims and extrapolations visibly separate.

## Depth Modes

- Cache-first: use the bundled references and label extrapolations.
- Latest-aware: refresh only the lanes exposed to current events or recent publications.
- Quote-backed: add URLs and short quotes where needed, staying within source limits.

If the host cannot browse or fetch, say what could not be verified.
