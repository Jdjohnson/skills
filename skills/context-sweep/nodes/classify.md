# Classify

Keep an item only when it is work-related, decision-relevant, likely to matter during later reconstruction, and meaningfully additive.

Skip trivial acknowledgements, low-signal chatter, pure logistics, repeated follow-ups with no new context, generated noise, and duplicates.

Write short clean summaries:

- no raw dumps
- no connector jargon
- no personal data that is unnecessary to the log
- no invented facts
- no false certainty when a source is partial

Put source limitations in the source-status note instead of repeating them in every bullet.
