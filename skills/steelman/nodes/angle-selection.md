## Prioritize By Idea Type

- Personal decision: motivation, assumptions, tradeoffs, day after.
- Business strategy: opposition, context, execution, tradeoffs.
- Argument or position: core, opposition, assumptions.
- Process or system: execution, assumptions, day after.
- Creative project: motivation, audience context, differentiation.

## Rule

Start with the highest-risk unknown. Do not run every angle mechanically.

If the user asks for a specific angle, honor it, then return to the highest-value remaining risk.
