# Local Guardrails

## Output Discipline

- Keep the receipt short and concrete.
- Separate active, filed, deferred, and blocked intake items.
- Do not blur drafted local work with sent, posted, or externally mutated work.
- Name exact files moved into `DOT_COMMS_HOME/runtime/jj-intake/` when relevant.

## Interaction Gates

- Ask before outbound sends, publishing, calendar invites, CRM mutations, or task creation.
- Treat refresh as local inbox preparation only; it is not permission to clean up real email or SMS inboxes.
- If an item needs user judgment, show the five-field format and wait.

## Shell And Secret Safety

- Do not print secrets, tokens, cookies, raw headers, or private credentials.
- Prefer existing helper scripts or documented commands over ad hoc shell work.
- Stop and report when the required local queue or workspace path is missing.
