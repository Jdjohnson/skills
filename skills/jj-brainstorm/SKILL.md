# JJ Brainstorm

Use this skill to keep thinking moving when an idea is still forming.

## Workflow

| Step | Purpose | Node |
|------|---------|------|
| 1 | Align on the topic and run the brainstorm loop | [workflow](nodes/workflow.md) |
| 2 | Keep questions useful and non-repetitive | [question quality](nodes/question-quality.md) |
| 3 | Maintain an optional routed scratch note | [scratch note](nodes/scratch-note.md) |
| 4 | Use short research detours when facts matter | [research detour](nodes/research-detour.md) |
| 5 | Wrap when the idea has shape | [wrap](nodes/wrap.md) |

## Rules

- Echo the user's starting point in plain language before reframing it.
- Help the user see options, tensions, assumptions, and next moves.
- Ask small sets of high-leverage questions.
- Do not force a plan before the idea is ready.
- Use a scratch note when the session becomes substantive or the user asks to capture the thinking.
- Ask before creating or updating the scratch note unless the user explicitly asked to save or capture.
- Follow `ROUTES.md` for every durable output.
