# Scratch Note

A scratch note is a living working-memory file for a substantive brainstorm. It is not a transcript.

Use it when:

- the user asks to capture the thinking
- the discussion has enough substance to be useful later
- the topic will likely continue across sessions

Suggested sections:

- Current State
- Pressure Points
- What We Learned
- Open Questions
- Research Findings
- Likely Next Moves

Route the scratch note to the nearest relevant project `dot/` folder or timeline day `dot/` folder. Ask before creating or updating it unless the user explicitly asked to save or capture.
