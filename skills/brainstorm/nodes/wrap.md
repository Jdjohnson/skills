## When To Suggest Wrapping

Suggest a wrap when:

- the topic has a clear shape
- new rounds are mostly repeating
- the user asks whether there is enough
- the next move is no longer brainstorming

Do not stop unilaterally. Ask for approval when the user has not already asked to wrap.

## Wrap Output

```markdown
## Clearest Version
[the idea as it now stands]

## Strongest Pressure Points
- ...

## Still Unresolved
- ...

## Best Next Move
[plan, steelman, research, pause, or continue]
```

Offer file output only if the user asks to save the summary or provides a destination.
