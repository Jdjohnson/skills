## Default Mix

Ask exactly three questions in an active round:

1. One framing question: what is this really about, or which outcome matters most?
2. One pressure question: what assumption is weak, or where is the contradiction?
3. One movement question: what choice, boundary, or next distinction would move the topic forward?

## Good Questions

- clarify what the user means, wants, or is optimizing for
- surface the real tension rather than circling the symptom
- force decision-relevant specificity
- keep the topic moving toward shape
- use a framework only when it helps the user answer faster

## Avoid

- already-answered questions
- obvious prompts that feel like filler
- sprawling multipart questions
- soft validation questions that do not change the topic
- clever side paths that do not improve the next decision
