## When To Propose Research

Propose a short detour when a factual gap would noticeably improve the next round.

Good reasons:

- real examples would sharpen the idea
- benchmarks or outside patterns would change the decision
- the user supplied a URL or file that should anchor the brainstorm
- a missing fact affects the shape, not just polish

## Detour Shape

1. Name the factual gap in one sentence.
2. Ask whether to take a short detour.
3. If approved, gather only what is needed.
4. Return to `Current State`, update `Pressure Points`, then ask the next three questions.

If the user declines, keep moving from the current evidence and mark the assumption.

## Source Discipline

- Use supplied files or URLs first.
- Use live lookup only when the host supports it and the fact matters.
- Do not let research become a separate report unless the user asks.
