# Brief Format

Use these sections unless the user requests a different format:

## Bottom Line

Two or three sentences with the main takeaway.

## Key Judgments

Three to five bullets. Tag each judgment `[High]`, `[Medium]`, or `[Low]` confidence when confidence matters.

## What To Decide

Name the decision, likely owner, and next action when the brief supports a decision. If there is no decision, use `What Matters`.

## Gaps

State what is unknown, weakly sourced, time-sensitive, or worth checking next.

Keep the brief short. If the topic needs more depth, offer a follow-up instead of stuffing the brief.
