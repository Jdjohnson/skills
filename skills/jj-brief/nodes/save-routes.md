# Save Routes

Follow `ROUTES.md`.

Common routes:

- brief created by Dot: nearest relevant `dot/`
- source material shared by the user: nearest relevant `source/`
- project-owned human context: project `human/` only when explicitly requested

Do not write briefs outside routed Dot folders.

Before saving, show the target path and ask for approval unless the user explicitly asked to save or capture the work.
