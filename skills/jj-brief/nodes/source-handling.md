# Source Handling

Accept:

- pasted text
- file paths
- links
- meeting notes
- project notes
- a topic or question

Use the source provided before researching. If the source is missing or too thin, ask one question or run a focused search when the user expects current facts.

For multiple sources, synthesize. Do not summarize each source one by one unless the user asks.

Keep citations or source references where they help the user trust the brief.
