# Workflow

Choose the lightest lane that can help:

| Lane | Use When | Output |
|------|----------|--------|
| `start` | The user has an idea, topic, or rough notes | A short writing plan and first useful questions |
| `draft` | The user wants a first pass now | Draft first, then brief assumptions |
| `revise` | The user provides existing text | Revised text or a targeted revision plan |

Default to `start` unless the user asks to draft or revise.

Keep momentum. If the user gives enough context, draft instead of over-interviewing. If the user gives too little context, ask one compact question that unlocks the draft.
