description: Decide how to gather and bound source material for a brief.
---

## Input Handling

Accept any of these inputs:

- Pasted text: analyze directly.
- File path: read the file if the host supports file access.
- URL: fetch or browse the source if the host supports it.
- Topic or question: do a focused research pass if search is available.
- Multiple sources: synthesize into one unified brief.

## Bound The Work

This skill is for a concise executive brief, not a comprehensive research report.

Use proportional effort:

- Supplied material: no extra research unless needed for context or verification.
- URL or file: brief the source and flag gaps.
- Topic or question: use a focused search pass when available.
- 10+ source investigation: recommend a deep research workflow instead.

## Rules

- Say when source access is unavailable.
- Separate supplied facts from judgments.
- Do not invent citations, owners, deadlines, quotes, or source details.
- If evidence is thin, keep confidence low and say why.
