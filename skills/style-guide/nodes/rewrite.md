## Rewrite Guidance

Load [[source-of-truth.md]] first, then run the red-flag logic from [[red-flags.md]] before rewriting.

## Rules

- Preserve facts, meaning, stance, and level of certainty.
- Prefer small edits when the draft is already close.
- Do not flatten strong opinions into neutral summary.
- Do not add polish that makes the piece less believable.
- Keep useful roughness when it serves the voice.
- Do not add new examples, claims, praise, citations, or deadlines unless the user provided them.
- If a sentence needs a missing concrete detail, either ask for it or remove the sentence.

## Output

```markdown
## Revised Copy

[copy]

## Style Notes
- ...
```

Keep the style notes short. The rewrite is the main product.
