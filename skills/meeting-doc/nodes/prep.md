## Prep

Use this when the user wants to get ready before a meeting.

If the supplied context is thin, ask for only the missing item that would change the brief: agenda, attendees, prior notes, desired outcome, or current concern.

## Output

```markdown
# Prep Brief: [Meeting Title]

## What This Is
[2-4 sentences]

## What Matters
- ...

## Talking Points / Decisions
- ...

## Context
- ...

## Watchouts
- ...
```

For routine meetings, keep the brief short and say when no meaningful watchout is visible.
