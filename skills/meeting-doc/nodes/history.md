## History

Use this when the user provides or points to prior meeting context and wants orientation.

Do not assume a durable meeting system exists. Work from supplied files, notes, transcripts, links, or summaries.

## Output

```markdown
# Meeting History: [Person, Account, Topic, Or Range]

## Relevant Prior Meetings
- [date/title if known]: [why it matters]

## What Carried Forward
- ...

## Decisions Or Commitments Still Relevant
- ...

## Open Threads
- ...
```

If several meetings matter, present the short list first, then synthesize.
