## Input Sources

Use the lightest useful source set:

1. context pasted in the conversation
2. files or excerpts the user points to
3. URLs the user provides, when fetching is supported
4. short guided debrief when the record is too thin

Do not assume the host has a calendar, transcript app, mail inbox, chat workspace, or meeting database.

## Clarifying Questions

Ask only when one of these is unclear:

- target meeting
- mode: prep, close, history, or review
- intended audience
- missing source material that would change the document

If the user gives enough to produce a useful artifact, proceed and name any assumptions briefly.
