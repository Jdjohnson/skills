## Phase 7: Refine

Run a final pass for:

- voice consistency
- factual accuracy
- argument clarity
- rhythm
- concrete detail
- ending strength
- signs of AI-generated prose

## Humanize Mode

When the user asks to humanize text:

- preserve meaning and stance
- cut padded openings
- replace generic abstractions with concrete language
- remove fake balance and over-explained transitions
- vary sentence rhythm
- keep useful roughness

Do not make the text more polished if polish makes it less believable.

## Output

```markdown
## Revised Copy

[copy]

## Notes
- ...
```

If no rewrite is needed, say so and name the few small edits you would make.
