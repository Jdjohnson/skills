## Phase 5: Structure

Propose a simple arc that fits the piece:

- hook or opening observation
- central claim
- supporting sections
- counterpoint, tension, or complication
- landing

Do not force a marketing template onto reflective, argumentative, or personal writing.

## Structure Options

Use the lightest structure that works:

- Short post: claim -> reason -> example -> landing.
- Essay/article: observation -> evidence/story -> thesis -> complication -> landing.
- Memo: context -> recommendation -> rationale -> risk -> next step.
- Talk: premise -> stakes -> examples -> practical takeaways -> close.

## Output

```markdown
## Proposed Structure

1. ...
2. ...
3. ...
```

If the user asked you to draft immediately, use this structure silently and include only a short note afterward.
