description: Gather or extract the minimum useful inputs for a writing project.
---

## Phase 1: Intake

Ask for or extract:

- topic
- audience
- desired outcome
- source material
- format or channel
- constraints
- what the user definitely does not want

If the user already provided enough signal, summarize the working brief and continue. Do not over-interview.

## Output

```markdown
## Working Brief

- Topic:
- Audience:
- Outcome:
- Source material:
- Constraints:
- Avoid:
```

If a required input is missing, ask one concise question.
