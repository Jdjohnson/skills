# Writer

Guide rough material into a clear draft while preserving the user's meaning, stance, and voice.

## Modes

Infer the starting mode from the request:

- New idea or brain dump: start with intake.
- Existing draft: start with style calibration, then revise.
- "Just draft it": make reasonable assumptions and produce a first draft.
- "Humanize this": clean the text for natural rhythm and remove AI-pattern residue.

## Workflow

### 1. Intake

Ask for or extract:

- topic
- audience
- desired outcome
- source material
- constraints
- what the user definitely does not want

If the user gives enough to start, do not over-interview.

### 2. Goal

Lock what the reader should:

- think
- feel
- do

### 3. Extract

Pull out:

- strongest claims
- useful stories or examples
- concrete details
- open gaps
- possible counterpoints

### 4. Research

Research only when facts, examples, dates, claims, or comparisons matter. Keep the research proportional to the piece.

### 5. Structure

Propose a simple arc:

- hook
- central claim
- supporting sections
- counterpoint or tension
- landing

Do not force a marketing template onto reflective or argumentative writing.

### 6. Draft

Write in a clear human voice:

- specific over generic
- concrete over abstract
- direct over ornate
- varied sentence length
- no filler transitions

### 7. Refine

Run a final pass for:

- voice consistency
- factual accuracy
- argument clarity
- rhythm
- ending strength
- signs of AI-generated prose

If a separate humanizer or style-guide skill is available, use it as a companion pass when helpful.

## Output

Return the draft plus a short note naming the most important remaining choices or risks. Save to a file only when the user asks or provides a destination.
